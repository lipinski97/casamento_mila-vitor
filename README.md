# Vitor & Camila — Convite de Casamento

Este projeto é um convite de casamento moderno e diferente, pensado como um presente especial para Camila e Vitor. A ideia é fugir do site tradicional e entregar uma experiência visual e interativa, com as principais informações do casamento, confirmação de presença e cotas de presente fictícias.

## Objetivo

- Apresentar o evento de maneira elegante e personalizada
- Mostrar local, igreja e dresscode
- Permitir confirmação de presença (RSVP)
- Exibir "cotas de presente" fictícias
- Indicar que o envio de dinheiro deve ser feito diretamente para o casal (Camila e Vitor)

## O que contém

- Seção de boas-vindas com destaque para o casal
- História de "Nossa Jornada"
- Informações do dia do casamento
- Cartões de cerimônia e recepção
- Formulário de RSVP
- Lista de presentes fictícios
- Rodapé com crédito estilizado

## Informações do casamento

- Local da cerimônia: Igreja Matriz de Curitiba
- Horário da cerimônia: 15 de Outubro de 2024, às 16h00
- Local da recepção: Castelo do Batel
- Dresscode: Esporte Fino

## Como usar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Abra o endereço exibido no terminal para visualizar o convite.

## Como presentear

As cotas de presente são fictícias e foram pensadas como sugestões criativas. Para presentear o casal, envie o valor diretamente para Camila e Vitor usando o meio de pagamento acordado entre vocês.

> Este site é uma forma especial de reunir as informações do casamento e celebrar o momento de forma original.

## Tecnologia

- React
- Vite
- Tailwind CSS
- Lucide icons

## Estrutura do projeto

- `index.html` — entrada principal do Vite
- `src/main.jsx` — inicialização do React
- `src/App.jsx` — componente principal do convite
- `src/index.css` — estilos base e importação de fontes
- `tailwind.config.js` — configuração do Tailwind
- `vite.config.js` — configuração do Vite

## Observação
